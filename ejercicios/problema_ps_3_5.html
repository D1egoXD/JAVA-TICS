Problema PS 3.5 - Suma de Serie Armónica

Volver al Menú Principal

Enunciado del Problema

Escriba un diagrama de flujo que lea un número entero N y calcule el resultado de la siguiente serie:

1 + 1/2 + 1/3 + 1/4 + ... + 1/N

Nota: Esta es una serie armónica parcial, donde cada término es el recíproco de un número natural. La suma crece logarítmicamente con N.

Explicación de las Variables

  • N: Variable de tipo entero que representa el número de términos de la serie (entrada del usuario).
  • I: Variable de tipo entero. Contador del ciclo (desde 1 hasta N).
  • SUMA: Variable de tipo real (double). Acumulador de la suma de la serie (resultado).

Diagrama de Flujo

El diagrama muestra un proceso iterativo que suma los términos 1/I para I desde 1 hasta N.

Nota: Si el diagrama no se muestra, ejecuta el script de Python para generar los diagramas:

python generate_diagrams.py

Ejemplos de Ejecución

A continuación podemos observar el cálculo de la serie para diferentes valores de N:

Caso 1: N = 4

Cálculo paso a paso:

  • Término 1: 1/1 = 1.0000
  • Término 2: 1/2 = 0.5000
  • Término 3: 1/3 = 0.3333
  • Término 4: 1/4 = 0.2500

Suma total: 1.0000 + 0.5000 + 0.3333 + 0.2500 = 2.0833

Caso 2: N = 1

Suma total: 1/1 = 1.0000

Caso 3: N = 10

Suma total: 1 + 1/2 + 1/3 + ... + 1/10 ≈ 2.9290

Implementación en Java (Programación Estructurada)

public class problema_ps_3_5 {
    
    /**
     * Calcula la suma de la serie armónica: 1 + 1/2 + 1/3 + ... + 1/N
     * @param n Número de términos de la serie
     * @return Suma de la serie
     */
    public static double calcularSerie(int n) {
        double suma = 0.0;
        
        for (int i = 1; i <= n; i++) {
            suma += 1.0 / i;
        }
        
        return suma;
    }
    
    public static void main(String[] args) {
        // Ejemplo de uso
        int n = 4;
        double resultado = calcularSerie(n);
        System.out.printf("La suma de la serie para N=%d es: %.4f%n", n, resultado);
    }
}

Actividades para el Estudiante

Archivo: problema_ps_3_5_Estudiante.java

Completa el método calcularSerie() siguiendo estos pasos:

  1. Inicializa la variable SUMA en 0.0
  2. Crea un ciclo for que vaya desde I=1 hasta I=N
  3. En cada iteración, suma 1.0/I a SUMA
  4. Retorna el valor de SUMA
